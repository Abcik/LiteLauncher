Privacy Statement

This page describes the privacy position of the public LiteLauncher project published by Abcik.

Contact: abcik@ukr.net

Summary

LiteLauncher is being prepared for broader release. Public project materials are limited to the launcher codebase, interface assets, and documentation included in this repository.

Data collection

The public repository and GitHub Pages website do not require users to create an account and do not intentionally collect personal data through the static website itself.

If authentication is added in a future approved release, this statement may be updated to describe what data is processed, why it is needed, and how it is handled.

Source hosting

This project is published through GitHub. GitHub may process technical data such as IP address, browser information, and access logs according to its own policies when someone visits the repository or the GitHub Pages site.

Future account support

LiteLauncher is intended to support standard Microsoft account authorization after platform approval. The public project does not present an unofficial offline or cracked account flow.

Changes

This statement may be updated as LiteLauncher development progresses and as platform-integrated functionality becomes available.
